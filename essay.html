Essay-Modul

Formulieren Sie auf dieser Seite einen literarischen Essay zu Der Ausflug der toten Mädchen. Das Modul folgt der Struktur These → Argument → Textbezug.

1. These

Formulieren Sie eine präzise, diskutierbare These zum Text.

2. Argument I

Entfalten Sie Ihr erstes Argument zur Stützung der These.

3. Textbezug I

Belegen Sie Argument I mit einer konkreten Textstelle.

4. Argument II

Entfalten Sie ein weiteres, vertiefendes Argument.

5. Textbezug II

Belegen Sie Argument II mit einer weiteren Textstelle.

6. Meta-Reflexion

Reflektieren Sie Ihr Vorgehen und Ihre Deutung.

Export

Exportieren Sie Ihren Arbeitsstand.
